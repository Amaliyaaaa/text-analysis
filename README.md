# text-analysis
Это небольшое python-приложение для сравнения двух текстов, созданное средствами библиотек python.
После анализа программа выдает результат, состоящий из двух параметров: 
1. Сходство текстов: отражает общую схожесть текстов.
2. Пересечение слов: показывает, какой процент слов, зафиксированных в тексте 1, присутствует в тексте 2.
Приложение можно использовать для проверки учебных текстов (статей, сочинений, эссе) на плагиат, для проверки соотвестствия текста шаблону и в других целях, связанных с проверкой текстовых данных на соответствие какому-либо правилу или стандарту.
Знаки препинания, регистр, пробелы и переносы в данной версии не учитываются. Анализ производится по каждому отдельномуу слову.
### Как протестировать:
Скачать и запустить exe-файл.
